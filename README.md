# SUMEC MiniSumo
Tento projekt se zaměřuje na design a konstrukci Mini sumo robota (10X10 cm, 500g).

## Součástky
### Akumulátor:
- LiPo 450mAH 75C 3S
### H-můstek:
- TB6612FNG
### Motory:
- 150 RPM, 12V x2
### Senzor čáry:
- QRE1113 x4
### Laserove senzory:
- VL53L0X x3
